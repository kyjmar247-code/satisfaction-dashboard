# 대시보드 설치 및 실행 가이드

## 1단계: Node.js 설치

### 방법 1: 공식 사이트에서 다운로드 (권장)

1. **Node.js 공식 사이트 접속**
   - 브라우저에서 https://nodejs.org 접속
   - 또는 직접 다운로드: https://nodejs.org/ko/download/

2. **다운로드**
   - "LTS" 버전 (장기 지원 버전) 다운로드
   - Windows Installer (.msi) 파일 선택
   - 예: `node-v20.x.x-x64.msi`

3. **설치**
   - 다운로드한 파일 실행
   - "Next" 버튼을 클릭하며 기본 설정으로 설치
   - "Add to PATH" 옵션이 체크되어 있는지 확인 (기본적으로 체크됨)

4. **설치 확인**
   - 명령 프롬프트(cmd) 또는 PowerShell을 새로 열기
   - 다음 명령어 입력:
   ```bash
   node --version
   npm --version
   ```
   - 버전 번호가 표시되면 설치 완료!

### 방법 2: Chocolatey 사용 (선택사항)

Chocolatey가 설치되어 있다면:
```bash
choco install nodejs-lts
```

## 3단계: 의존성 설치

프로젝트 폴더에서 다음 명령어 실행:

```bash
npm install
```

**설치 과정:**
- 인터넷 연결 필요
- 약 1-2분 소요
- `node_modules` 폴더가 생성됨
- "added XXX packages" 메시지가 나오면 완료

**오류 발생 시:**
- 인터넷 연결 확인
- 방화벽/보안 프로그램 확인
- 관리자 권한으로 실행 시도

## 5단계: 개발 서버 실행

프로젝트 폴더에서 다음 명령어 실행:

```bash

```

- "Local: http://localhost:3000" 메시지 표시
- 브라우저가 자동으로 열림

- 터미널에서 `Ctrl + C` 누르기

## 문제 해결

### "node: command not found" 오류
- Node.js가 설치되지 않았거나 PATH에 추가되지 않음
- Node.js 재설치 또는 컴퓨터 재시작

### "npm: command not found" 오류
- Node.js 설치 시 npm이 함께 설치되지 않음
- Node.js 재설치

### "Cannot find module" 오류
- `npm install`을 실행하지 않음
- `npm install` 실행 후 다시 시도

### CSV 파일을 찾을 수 없다는 오류
- `dashboard/public/converted/` 폴더에 CSV 파일이 있는지 확인
- 파일명이 정확한지 확인 (특수문자, 공백 포함)

### 포트 3000이 이미 사용 중
- 다른 프로그램이 3000번 포트 사용 중
- `vite.config.js`에서 포트 번호 변경 가능
