# 대시보드 배포 가이드

## 📦 배포 준비

### 1. 빌드 실행
`빌드_및_배포.bat` 파일을 더블클릭하여 실행하세요.

이 파일은 다음 작업을 수행합니다:
- 의존성 설치
- 프로젝트 빌드
- CSV 파일 복사

### 2. 배포 폴더 확인
빌드가 완료되면 `dist` 폴더가 생성됩니다. 이 폴더 전체를 사업팀에 전달하세요.

```
dist/
├── index.html
├── assets/
├── converted/
│   ├── 2025 하반기 재원생 만족도조사 데이터 및 문항속성_챗GPT_데이터.csv
│   └── 2025 하반기 재원생 만족도조사 데이터 및 문항속성_챗GPT_문항 및 보기.csv
└── 대시보드_실행.bat
```

---

## 🚀 사업팀 사용 방법

### 방법 1: 배치 파일 실행 (권장)
1. `dist` 폴더를 받은 후
2. `대시보드_실행.bat` 파일을 더블클릭
3. 브라우저에서 `http://localhost:8000` 접속

### 방법 2: Node.js 사용 (Python이 없는 경우)
1. Node.js가 설치되어 있어야 합니다
2. `dist` 폴더에서 명령 프롬프트 열기
3. 다음 명령 실행:
   ```bash
   npx serve -s . -l 3000
   ```
4. 브라우저에서 `http://localhost:3000` 접속

### 방법 3: 간단한 웹 서버 사용
다른 웹 서버 도구를 사용할 수도 있습니다:
- **Live Server** (VS Code 확장)
- **XAMPP**
- **IIS** (Windows)

---

## ⚠️ 주의사항

1. **CSV 파일 경로**: `converted` 폴더의 CSV 파일이 반드시 있어야 합니다.
2. **브라우저**: 최신 버전의 Chrome, Edge, Firefox 권장
3. **포트 충돌**: 8000번 포트가 사용 중이면 다른 포트를 사용하세요.
   - Python: `python -m http.server 8080`
   - Node.js: `npx serve -s . -l 8080`

---

## 📋 시스템 요구사항

- **운영체제**: Windows 10 이상
- **Python**: 3.x (방법 1 사용 시)
- **또는 Node.js**: 14.x 이상 (방법 2 사용 시)
- **브라우저**: Chrome, Edge, Firefox 최신 버전

---

## 🔧 문제 해결

### 서버가 시작되지 않는 경우
- 방화벽 설정 확인
- 다른 포트 사용 시도
- 관리자 권한으로 실행

### 데이터가 표시되지 않는 경우
- CSV 파일이 `converted` 폴더에 있는지 확인
- 브라우저 콘솔(F12)에서 오류 확인
- CSV 파일 이름이 정확한지 확인
