# 사업팀 전달 가이드

## 📦 전달할 폴더

**`dist` 폴더 전체**를 사업팀에 전달하세요.

## 📁 dist 폴더 구조

```
dist/
├── index.html                                    (메인 HTML 파일)
├── assets/                                       (자바스크립트, CSS 파일)
│   ├── index-xxxxx.js
│   └── index-xxxxx.css
├── converted/                                    (CSV 데이터 파일 - 필수!)
│   ├── 2025 하반기 재원생 만족도조사 데이터 및 문항속성_챗GPT_데이터.csv
│   └── 2025 하반기 재원생 만족도조사 데이터 및 문항속성_챗GPT_문항 및 보기.csv
├── 대시보드_실행.bat                              (실행 파일 - 더블클릭)
└── README_사용법.txt                             (사용 가이드)
```

## ✅ 전달 전 확인사항

1. **CSV 파일 확인**: `dist/converted/` 폴더에 CSV 파일 2개가 있는지 확인
2. **실행 파일 확인**: `dist/대시보드_실행.bat` 파일이 있는지 확인
3. **모든 파일 포함**: `dist` 폴더 전체를 압축하거나 전달

## 🚀 전달 방법

### 방법 1: 폴더 전체 전달
- `dist` 폴더를 그대로 전달
- 폴더명을 변경해도 됩니다 (예: "만족도조사_대시보드")

### 방법 2: ZIP 압축 파일로 전달
- `dist` 폴더를 ZIP으로 압축
- 파일명: `만족도조사_대시보드.zip`
- 사업팀에서 압축 해제 후 사용

## 📋 사업팀 사용 방법

1. `dist` 폴더를 받은 후
2. `대시보드_실행.bat` 파일을 더블클릭
3. 브라우저에서 `http://localhost:8000` 접속

## ⚠️ 중요 사항

- **CSV 파일은 반드시 포함되어야 합니다**
- **모든 파일을 함께 전달해야 합니다** (일부만 전달하면 작동하지 않음)
- **폴더 구조를 유지해야 합니다** (converted 폴더 위치 중요)

## 🔧 빌드 방법

전달 전에 반드시 빌드를 실행하세요:

1. `빌드_및_배포.bat` 파일을 더블클릭
2. 빌드가 완료되면 `dist` 폴더 확인
3. 위의 파일들이 모두 있는지 확인 후 전달
