# 2025 하반기 재원생 만족도 조사 대시보드

Vue3 + Vite + Chart.js를 사용한 인터랙티브 만족도 조사 대시보드

## 설치 및 실행

### 1. Node.js 설치
Node.js가 설치되어 있지 않다면 [Node.js 공식 사이트](https://nodejs.org/)에서 설치하세요.

### 2. 의존성 설치
```bash
cd dashboard
npm install
```

### 3. CSV 파일 준비
`dashboard/public/converted/` 폴더에 다음 CSV 파일들을 복사하세요:
- `2025 하반기 재원생 만족도조사 데이터 및 문항속성_챗GPT_데이터.csv`
- `2025 하반기 재원생 만족도조사 데이터 및 문항속성_챗GPT_문항 및 보기.csv`

또는 `dashboard/public/` 폴더에 `converted` 심볼릭 링크를 만들어 사용할 수 있습니다.

### 4. 개발 서버 실행
```bash
npm run dev
```

브라우저에서 http://localhost:3000 으로 접속하세요.

### 5. 프로덕션 빌드
```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

## 주요 기능

### 1. 필터 시스템
- **권역/그룹/지점 필터**: 계층적 필터링
  - 권역 선택 시 해당 권역의 그룹만 표시
  - 그룹 선택 시 해당 그룹의 지점만 표시
  - "전체" 선택 시 상위 필터만 적용

### 2. 핵심 지표 (4개)
- **전체 만족도 평균**: 5점 척도 평균
- **추천 의향**: 추천 관련 문항 평균
- **재등록 의향**: 재등록 관련 문항 평균
- **불만족 비율**: 불만족 응답 비율

각 지표는 **전체 / 권역 / 그룹 / 지점** 4개 레벨로 비교 표시됩니다.

### 3. 불만족 주관식 답변 대시보드
- 불만족 응답자의 자유 응답 텍스트 수집
- 문항별 필터링
- 키워드 검색
- 페이지네이션

## 데이터 구조 설정

### 권역/그룹 매핑
`src/utils/dataLoader.js` 파일의 `locationMapping` 객체를 실제 데이터에 맞게 수정하세요:

```javascript
const locationMapping = {
  '강남': { region: '서울', group: '강남권' },
  '분당정자': { region: '경기', group: '남부권' },
  // ... 실제 지점명과 권역/그룹 매핑
}
```

### 메트릭 계산 로직
`src/utils/metrics.js` 파일에서 실제 문항 코드에 맞게 수정하세요:
- 전체 만족도: `S3 Q3` (전반적 만족도)
- 추천 의향: `S1 Q9` 또는 실제 문항 코드
- 재등록 의향: `S7` 섹션의 해당 문항

## 커스터마이징

### 스타일
`src/style.css` 파일에서 색상, 폰트, 레이아웃을 수정할 수 있습니다.

### 차트
Chart.js를 사용한 차트 컴포넌트를 추가하려면 `vue-chartjs` 라이브러리를 활용하세요.

## 문제 해결

### CSV 파일을 찾을 수 없다는 오류
- `dashboard/public/converted/` 폴더에 CSV 파일이 있는지 확인하세요
- 파일명이 정확한지 확인하세요 (특수문자, 공백 포함)

### 권역/그룹이 제대로 표시되지 않음
- `src/utils/dataLoader.js`의 `locationMapping`이 실제 지점명과 일치하는지 확인하세요
- 지점명이 데이터의 `지점` 컬럼과 정확히 일치해야 합니다
